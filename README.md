## Getting started

1. Install dependencies.

    ```bash
    npm i
    ```

2. Start the development server.

    ```bash
    npm run dev
    ```

3. Build for production.

    ```bash
    npm run build
    ```

4. Test your production build.

    ```bash
    npm run serve
    ```
